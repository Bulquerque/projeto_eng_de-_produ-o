# Entendimento do desafio

O projeto pede um simulador de malha logística capaz de comparar cenários para duas empresas diferentes.
O foco é verificar aderência ao cenário base, qualidade dos dados, decomposição de custos, comparação de cenários e apoio à decisão.

O simulador deve responder: dado um cenário de malha, quais são os custos logísticos, efeitos operacionais, riscos e eventuais impactos tributários, sempre comparando contra o cenário base correto da empresa analisada.
