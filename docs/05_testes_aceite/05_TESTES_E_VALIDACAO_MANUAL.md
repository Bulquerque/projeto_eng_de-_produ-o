# Testes e validação manual

Os testes por fase estão em `data/validation/phase_tests.json`.

Além dos testes automáticos, o usuário deve conseguir abrir cada página de fase e verificar visualmente:

- os dados carregados;
- a empresa selecionada;
- as premissas usadas;
- o baseline;
- os cenários;
- os alertas;
- o ranking;
- os resultados exportáveis.
