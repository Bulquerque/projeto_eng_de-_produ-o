# Documentação organizada

A documentação foi separada por tema para facilitar manutenção e leitura.

## Ordem recomendada de leitura

1. `00_inicio/` — índice e leitura inicial.
2. `01_visao_geral_do_projeto/` — entendimento do desafio, arquitetura e regras de separação das empresas.
3. `02_fases_do_projeto/` — plano por fase e guias da Fase 1.
4. `03_modulos_contratos_funcoes/` — módulos, inputs/outputs, funções internas, dependências e testes.
5. `04_dados_caminhos_auditoria/` — dados, caminhos, auditoria e dicionário.
6. `05_testes_aceite/` — testes e critérios de aceite.
7. `06_manifestos/` — manifestos de documentação.

8. `../phases/` — documentação por feature/módulo.

Debug: abra `/debug/`.
