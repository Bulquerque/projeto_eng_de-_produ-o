# Observações da Transcrição - Visagio Challenge

## Calibração e Realismo
- **Pareamento com o Mundo Real:** A simulação deve ser "nivelada" ou pareada com os dados reais do cenário base para garantir confiabilidade (Calibração do Cenário Base).

## Comparação de Cenários
- **Comparação Multi-Instância:** A interface deve permitir a comparação simultânea de $N$ instâncias/cenários para análise de trade-offs.

## Customização de Otimização
- **Meta-heurística Flexível:** Implementar um simulador onde o usuário possa descrever sua própria função objetivo.
- **Algoritmo Sugerido:** Utilizar uma meta-heurística simples, como **Iterated Greedy Search**, para maximizar a função objetivo definida pelo usuário.
